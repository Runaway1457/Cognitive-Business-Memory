# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-20

### Added
- 🧠 **Cognitive Business Memory** - Initial release
- 🔍 **AI Query Interface** - Natural language queries with causal chain analysis
- 🌐 **3D Knowledge Graph** - Interactive Three.js visualization
- 📊 **Timeline Component** - Chronological event display
- 📈 **Statistics Dashboard** - Real-time metrics and alerts
- 🔗 **Causal Chain Detection** - Automatic connection tracing
- 📁 **Mock Data System** - Realistic enterprise data simulation

### Features
- 34 people entities (CEO, CTO, developers, sales, CS...)
- 7 client entities (NeoBank, Saúde Tech, Varejo Plus...)
- 42 data sources (Slack, Email, Jira, Meetings, Metrics...)
- 32 graph connections (causal, temporal, relational, semantic)
- Timeline spanning Oct 2024 - Jan 2026
- AI-powered query processing with z-ai-web-dev-sdk

### Tech Stack
- Next.js 16.1.1 with App Router
- React 19
- TypeScript 5
- Three.js + React Three Fiber
- Tailwind CSS 4
- shadcn/ui components
- Zustand state management

- Ricardo Mendes departure → Knowledge concentration risk
- Budget freeze → Resource constraint cascade

- Production incident → Churn trigger
- Recovery plan → Success story
